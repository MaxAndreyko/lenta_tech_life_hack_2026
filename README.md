# Lenta Tech Life Hack 2026

Система для распознавания ценников на видео снятого роботом с полок магазина.

## Структура

```text
backend/ <--- Модуль бэкенда: обработка видео, запуск моделей, формирование CSV
  app/ <--- Основной код бэкенда
    core/ <--- Конфигурация, настройки путей и logging
    pipelines/ <--- Сценарии запуска отдельных этапов пайплайна
    services/ <--- Бизнес-логика модулей обработки видео
    schemas/ <--- Dataclass-схемы и структуры данных
    tracking/ <--- Модуль трекинга с реализацией алгоритмов трекинга
    utils/ <--- Общие вспомогательные функции
  configs/ <--- YAML-конфиги backend-сервисов
    tracking.yaml <--- Выбор tracker-а и параметры трекинга
  data/ <--- Локальное хранилище входных данных и результатов
    videos/ <--- Входные видео для обработки
    frames/ <--- Извлеченные кадры и metadata.csv
    crops/ <--- Вырезанные области ценников для будущих этапов
    outputs/ <--- Финальные результаты, включая CSV
    temp/ <--- Временные файлы пайплайна
  logs/ <--- Локальные .log-файлы приложения
  tests/ <--- Автотесты backend-модулей
  requirements.txt <--- Python-зависимости проекта
```
